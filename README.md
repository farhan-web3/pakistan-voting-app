# 🇵🇰 Pakistan Voting App

A simple and interactive voting application built using HTML, CSS, and JavaScript. This project simulates a digital voting system and was developed as part of my Human-Computer Interaction (HCI) course.

## 🚀 Features
- User-friendly interface
- Real-time vote counting
- Data stored using browser local storage
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

## 🎯 Purpose
The main goal of this project is to apply HCI principles and create a smooth and intuitive user experience.

## 📌 Future Improvements
- Add an authentication system
- Connect with database (MongoDB)
- Improve UI/UX design
